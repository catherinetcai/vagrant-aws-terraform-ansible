# Vagrant AWS/Terraform

### Prereqs
- Vagrant
- Virtualbox
- Git 

### Install Details
Running: Ubuntu 16.04 - Xenial

Installs:
- Unzip
- Pip3
- AWS Cli
- Terraform

Default user: ubuntu

### Allowing AWS ENV passthrough

This is configured to allow for AWS ENV passthrough. Set your aws credentials up via env vars and they'll be allowed through.
